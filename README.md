# 发票管理系统（浏览器版）

基于 HTML + JavaScript 的纯前端发票管理工具，支持 PDF 电子发票自动解析、多主体收票方校验、重复检测、报销分类管理、Excel 导出等功能。

**无需安装 Python，无需启动服务器，双击 HTML 文件即可在浏览器中使用。**

## 功能特性

- **PDF 批量导入**：支持拖拽上传，自动解析发票字段
- **自动字段提取**：发票类型、号码、日期、购买方/销售方信息、金额、明细等
- **多主体收票方校验**：可配置多个收票方主体，导入时自动匹配名称和税号
- **异常标红**：重复发票、名称不符、税号不符独立标红
- **自定义字段**：报销分类、报销人、报销月、备注
- **筛选排序**：按报销月、报销人、发票类型、收票主体、发票号筛选
- **Excel 导出**：全部导出或勾选导出
- **数据备份**：一键导出 JSON 备份文件

## 环境要求

- 现代浏览器（Chrome / Edge / Safari / Firefox）
- 需要联网（加载 pdf.js、SheetJS、Dexie.js 等 CDN 资源）

## 使用方式

### 方式一：直接打开（推荐）

1. 解压文件包
2. 双击 `index.html`
3. 浏览器自动打开，即可使用

### 方式二：本地服务器（可选）

如需离线使用，可先把 CDN 资源下载到本地，然后用本地服务器打开：

```bash
# Python
cd 发票管理系统
python -m http.server 8080

# Node.js
cd 发票管理系统
npx serve
```

浏览器访问 `http://localhost:8080`

## 使用流程

### 1. 配置收票方主体

点击右上角「系统设置」→ 添加收票方主体：

| 公司名称 | 纳税人识别号 |
|---------|-------------|
| 示例公司 | 9144XXXXXXXXXXXX |

- 可添加多个主体
- 导入发票时，系统会自动比对购买方名称和税号
- 不匹配的发票会在列表中标红提示

### 2. 导入发票

点击「导入发票」→ 拖拽或选择 PDF 文件：
- 支持批量上传
- 仅接受 PDF 格式
- 导入后自动解析并校验

### 3. 查看与编辑

列表中异常发票会标红显示：
- **重复**：发票号码已存在
- **名称不符**：购买方名称与配置的收票方主体不匹配
- **税号不符**：购买方税号与配置的收票方主体不匹配

点击「编辑」可填写：
- 报销分类（自动提示历史分类）
- 报销人
- 报销月（格式 YYYY-MM）
- 备注

### 4. 筛选

支持以下筛选条件：
- 报销月
- 报销人
- 发票类型（下拉框自动加载已有类型）
- 收票主体（购买方名称）
- 发票号

### 5. 导出与备份

- **导出 Excel**：点击「导出 Excel」下载全部数据；先勾选发票再点击可导出选中部分
- **数据备份**：在「系统设置」中点击「备份数据库」，下载 JSON 备份文件

## 数据说明

| 说明 |
|------|
| 所有数据存储在浏览器的 IndexedDB 中，不上传任何服务器 |
| 换电脑或换浏览器时数据不会自动同步 |
| 建议定期使用「备份数据库」功能导出 JSON 文件 |
| 恢复数据：目前需要手动导入 JSON（后续版本支持） |

## 常见问题

**Q: 双击 index.html 后页面空白或报错？**  
A: 请使用 Chrome、Edge、Safari 或 Firefox 等现代浏览器。部分旧版浏览器不支持 IndexedDB。

**Q: 导入 PDF 后某些字段为空？**  
A: 部分 PDF 为图片型发票（扫描件），无法提取文字。目前仅支持文字型 PDF。

**Q: 如何迁移数据到另一台电脑？**  
A: 在「系统设置」中点击「备份数据库」导出 JSON 文件，在新电脑上打开系统后手动导入（后续版本支持自动导入）。

**Q: 清除浏览器数据后发票会丢失吗？**  
A: 会。IndexedDB 数据与浏览器绑定，清除浏览数据会导致发票丢失。请务必定期备份。
